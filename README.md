# Kütüphane Yönetim Sistemi

Bu proje Python ve Nesne Yönelimli Programlama (OOP) kullanılarak geliştirilmiş basit bir kütüphane yönetim sistemidir.

## Özellikler

- Kitap ekleme
- Kitap silme
- Kitap güncelleme
- Kitap listeleme
- Dergi ekleme
- Dergi silme
- Dergi güncelleme
- Dergi listeleme

## Kullanılan Teknolojiler

- Python
- OOP (Object Oriented Programming)
